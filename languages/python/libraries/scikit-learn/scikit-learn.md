# [Scikit-learn](http://scikit-learn.org/stable/index.html)

Is a [[python]] library for [[machine learning]]. It provides high-performance, easy-to-use data structures and data analysis tools.

## Installation

```bash
pip install scikit-learn
```

## Usage

```python
import sklearn  # This is the main library
import sklearn.datasets  # This module allows you to load datasets
import sklearn.linear_model  # This module allows you to use linear models
import sklearn.model_selection  # This module allows you to split data into training and testing sets
import sklearn.neighbors  # This module allows you to use k-nearest neighbors
import sklearn.tree  # This module allows you to use decision trees
import sklearn.ensemble  # This module allows you to use random forests
import sklearn.metrics  # This module allows you to calculate metrics for your models
import sklearn.preprocessing  # This module allows you to preprocess data
import sklearn.pipeline  # This module allows you to create a pipeline of preprocessing and modeling steps
import sklearn.feature_selection  # This module allows you to select features
import sklearn.svm  # This module allows you to use support vector machines
import sklearn.naive_bayes  # This module allows you to use naive bayes
import sklearn.cluster  # This module allows you to use clustering algorithms
import sklearn.manifold  # This module allows you to use manifold learning algorithms
import sklearn.decomposition  # This module allows you to use decomposition algorithms
import sklearn.neural_network  # This module allows you to use neural networks
import sklearn.gaussian_process  # This module allows you to use gaussian processes
import sklearn.discriminant_analysis  # This module allows you to use discriminant analysis
import sklearn.isotonic  # This module allows you to use isotonic regression
import sklearn.semi_supervised  # This module allows you to use semi-supervised learning
import sklearn.mixture  # This module allows you to use mixture models
import sklearn.covariance  # This module allows you to use covariance estimation
import sklearn.cross_decomposition  # This module allows you to use cross decomposition
import sklearn.calibration  # This module allows you to use calibration
import sklearn.multioutput  # This module allows you to use multioutput estimators
import sklearn.impute  # This module allows you to use imputation of missing values
import sklearn.compose  # This module allows you to use column transformer
import sklearn.experimental  # This module allows you to use experimental features
import sklearn.feature_extraction  # This module allows you to use feature extraction
```
