# [OpenCV](https://opencv.org/)
Is a [[python]] library for [[computer vision]]. It provides a high-performance multidimensional array object, and tools for working with these arrays.

## Installation
```bash
pip install opencv-python
```

## Usage
```python
import cv2 as cv
```
