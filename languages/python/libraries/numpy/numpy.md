# [Numpy](https://numpy.org/)

Is a [python](./../../python.md) library for scientific computing. It provides a high-performance multidimensional array object, and tools for working with these arrays.

## Installation
```bash
pip install numpy
```

## [Usage](numpy.ipynb)
```python
import numpy as np
```
