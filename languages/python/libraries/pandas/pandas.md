# [pandas](https://pandas.pydata.org/)
Is a [[python]] library for data analysis. It provides high-performance, easy-to-use data structures and data analysis tools.

Pandas is built on top of [[numpy]].

## Installation
```bash
pip install pandas
```

## Usage
```python
import pandas as pd
```
